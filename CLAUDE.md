# CLAUDE.md — project rules for AwesomeSecurityResearch

## Commits & PRs

- **No AI attribution.** Do NOT add `🤖 Generated with Claude Code`, `Co-Authored-By: Claude`,
  or any similar "generated by" / "co-authored by AI" line to commit messages or PR
  bodies — in this repo or in anything the automation opens. Write commits and PRs as
  plain, human-authored text.
- Conventional commit messages (`feat:`, `fix:`, `chore:`, `docs:`, `content:`).
- Never commit secrets or transient data: cookies, `.env`, `data/candidates.json`,
  `data/_raw/`, `data/analysis_out.json` stay gitignored.

See [AGENTS.md](AGENTS.md) for the full operating manual (pipeline, tests, daily automation).
